== 演習の概要
このモジュールでは、AROクラスターのコンソールに表示される、各プロジェクトのメトリクスデータを確認します。

=== 演習の手順
下記のWebページを参照して、演習を実行してください。
ROSAクラスター上の手順となっていますが、AROクラスターでも同様の手順で実行できます。

* https://rh-open.github.io/ocp-handson/openshift-services/55-3-rosa-project-metrics.html[ROSAクラスター上でのプロジェクトのメトリクスデータの確認]
