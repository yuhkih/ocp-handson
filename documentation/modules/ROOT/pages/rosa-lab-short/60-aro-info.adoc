Azure Red Hat OpenShift (ARO) Lab プロジェクトは、
インストラクター主導のデモ紹介またはセルフペースの演習で、AROを効果的に紹介及び体感していただくことを目的としています。


[Note]
====
AROの概要について学習する場合、
コンテナやOpenShift、AROの特徴やサービス仕様をご紹介している資料を参考にしてください。

* https://rh-open.github.io/ocp-handson/openshift-services/_images/aro/pdf/2024-aro-workshop-lecture.pdf[コンテナ/OpenShift/AROの紹介資料(PDFファイル)]
====


---

== 演習の前提要件

* Azureにアクセス可能なネットワーク環境
* AROクラスターにアクセス可能なWebブラウザ

NOTE: https://access.redhat.com/articles/4763741[「Browsers and Client Tools」表]にある、
最新版のOpenShiftに対応したWebブラウザ(Firefox/MS Edge/Chrome/Safari)のいずれかを利用します。


== 演習の進め方

* 本Labをワークショップ形式で実施する場合、[デモ]と分類している項目の内容は実施せずに、
そのまま読み進めて次の項目に進んで下さい。これらの内容は、すでに適用済みのAROクラスターをご利用いただきます。
AROクラスターのアクセス情報は、インストラクターがご案内します。

* 本Labを自習形式で実施する場合、全てのコンテンツを実施してください。

[Note]
====
次のものを用意することで、このLabを自習できます。

* 有償サービスを利用可能なAzureアカウント
* 無料で作成可能な
https://cloud.redhat.com/[Red Hatアカウント] ( `Create an account` から作成)
====
